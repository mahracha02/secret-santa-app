# Description de site
Un site web qui permet à un groupe d'organiser leur Secret Santa en ligne.
Chaque participant peut s'inscrire et voir (de manière confidentielle) le nom de la personne à qui il doit offrir un cadeau.

# Fonctionnalités de site
 
1.Page d’accueil  : avec bouton «  Commencer » pour lancer le processus
2.Formulaire d'inscription : noms des participants
3.Génération de paires : générer les paires aléatoirement
4.Résultats avec affichage sécurisé: lorsqu’on clique sur un nom, affichage de la personne à qui il doit offrir un cadeau
